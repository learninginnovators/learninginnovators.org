If you have Administration access, when you select the badge icon on the topr right corner you will see a drop down menu with three options.

### Manage Units 

This is covered in the Admin page.

### Manage Users

Here you can change the access level of other users.

Each user occupies a row, if you select a user's row you will see their details and be able to change their access level.

<img width="962" alt="Screenshot 2025-03-24 at 8 32 20 pm" src="https://github.com/user-attachments/assets/39bbde60-ec3b-483b-a4e4-13e79cd7dbb2" />


<img width="848" alt="Screenshot 2025-03-24 at 8 33 12 pm" src="https://github.com/user-attachments/assets/65b33401-310b-46aa-aa93-5d4c672c10ad" />

There are four access levels:

1. Admin - can manage units, users, and institutional settings
2. Convenor - can manage units
3. Tutor - can teach in units
4. Student - can be enrolled as a learner in units
   
<img width="296" alt="Screenshot 2025-03-24 at 8 32 08 pm" src="https://github.com/user-attachments/assets/0d9b8f97-034b-4af7-b57e-eb6601c5276a" />

### Institution settings

Here you can create new teaching periods eg. trimesters, and copy units from one trimester to another.

To crete a new teaching period select the tab "Teaching Periods" and select the blue icon labeled "Add new teaching period" on the top right. Fill the blank spacer with the ppropriate informtion and select "Create".



<img width="967" alt="Screenshot 2025-03-24 at 8 44 19 pm" src="https://github.com/user-attachments/assets/04a0f461-d94e-4dd6-beda-23981e9231fd" />

To copy units (roll over) to one teaching period, select the square and arrow icon on top of the pencil icon in the teching period you want to add units to. 

<img width="967" alt="Screenshot 2025-03-24 at 8 39 59 pm" src="https://github.com/user-attachments/assets/3846ae11-548e-46e6-81aa-3f3385721a5b" />

You will then be able to import a unit from any past teaching period. Follow the instructions carefully.
